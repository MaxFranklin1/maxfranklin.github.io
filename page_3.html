Comparing Jobs

Comparing a web development job to a games development job

A web developers job is design web pages for the internet, as well as making it look appealing enough to attract and retain users. A game developers job is to work with a massive team to slowly produce a product over 3 years. Web developers can also work in a team but nowhere near the same size. A game’s dev team is an entire company full of hundreds of employees all with difference skills and talents. The development schedule for games is much longer than we development. A game takes years to create where as a website might take around a month depending on the complexity of the website.

Both jobs demand attention to the product even after the products release. With a website a web dev must make sure everything is working as intended when it gets released to the masses. Unforeseen problems can arise from rapid increase of traffic so web developers need to be ready if something goes wrong. Same goes for game developers somewhat. A game still needs attention after release to fix unforeseen bugs and glitches. Game developers also start working on other projects almost immediately whereas for web developers the turnover rate between projects is a bit longer.

Once a web developer and its team gets a new project a large amount of time needs to be spent on pre-production and planning. Game developers also do this but it’s mostly done by other people and production is normally ready to begin as soon as people are finished with the last project.
